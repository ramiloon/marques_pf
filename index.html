Variables y tipos

Las variables guardan información. 

let nombre = "Ana";
let edad = 18;

Se guarda un nombre y una edad.

Operadores

Sirven para hacer operaciones. 

let a = 5;
let b = 3;

console.log(a + b);

Se suman dos números.

Condicionales

Permiten tomar decisiones. 

let nota = 7;

if(nota >= 5){
    console.log("Aprobado");
}

Si la nota es mayor que 5 aparece aprobado.

Bucles

Repiten código varias veces. 

for(let i = 1; i <= 5; i++){
    console.log(i);
}

Muestra números del 1 al 5.

Arrays

Guardan varios datos. 

let frutas = ["manzana", "pera"];

El array tiene dos frutas.

Funciones

Sirven para reutilizar código. 

function saludar(){
    console.log("Hola");
}

La función muestra un saludo.
